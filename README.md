# Rodando o projeto

1. Clone o repositório
2. Crie o ambiente virtual:
```console
py -3 -m venv venv
```
3. Ative o ambiente virtual:
```console
venv\Scripts\activate
```
4. Instale as dependências:
```console
pip install -r requirements.txt
```
5. Vá para a pasta src:
```console
cd src
```
6. Execute o flask:
```console
flask run
```
